# File Update VETERRAIN - Play Demo 3D

Isi ZIP ini adalah file pengganti:

```text
index.html
css/style.css
js/app.js
```

Fitur tambahan:
- Tombol `Play Demo Jalan 3D` setelah rute berhasil dihitung.
- Shortcut keyboard `R` untuk reset view.
- Rotasi 3D Map dibuat lebih nyaman dengan max pitch lebih tinggi.

Cara pakai:
1. Extract ZIP.
2. Copy `index.html`, folder `css`, dan folder `js` ke repo `VETERRAIN`.
3. Replace file lama.
4. Commit + push ke GitHub.
5. Setelah Pages update, tekan `Ctrl + F5` di browser.
